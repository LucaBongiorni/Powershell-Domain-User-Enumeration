Simple script for listing users in a domain, one user per line in a text file.

### USAGE:
Command Prompt*:

-`powershell -exec bypass -file User_Enumeration.ps1`

*Make sure you are running CMD as Admin.

Powershell:

-`.\User_Enumeration.ps1`
